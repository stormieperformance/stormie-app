What Stormiē is

Stormiē is an experimental internal tools lab focused on game development workflows and live service operations.

The platform brings together lightweight prototype applications designed to explore production workflows, content systems, and LiveOps incident handling.

Current tools

  • FeatureForge — game feature breakdown tool for structuring ideas into user stories, acceptance criteria, and MVP scope
  • FeatureScope — scope and complexity analysis tool for evaluating feature size, risk, dependencies, and delivery difficulty
  • Feature Dependency Map — prototype planning tool for visualizing how features connect to systems, teams, dependencies, and risks
  • Game Asset Similarity Finder — prototype tool for discovering replacement assets using semantic and visual similarity
  • LiveOps Incident Console — prototype operations tool for detecting incidents, routing issues to the correct teams, and tracking ownership across live service workflows

Built by Storm Kolmodin.

Tools are experimental prototypes exploring production workflows for modern game development.
